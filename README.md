# Call-Centre-Data-Analysis (Interactive Dashboard creation using MS Excel)
## Project Objective
The dashboard is designed to help stakeholders monitor and analyze various aspects of call center activities, identify trends, and make data-driven decisions to enhance efficiency and customer satisfaction in the 2023. 
## Data used
- <a href= "https://github.com/Vaishnavij2024/Data-analysis---Dashboard/blob/main/Call%20center%20Data.xlsx">Dataset</a>
## Questions(KPIs)
- What is the total number of calls and the total amount generated for the given period
- What is the average customer rating
- Compare the number of male callers to the number of female callers
- Which month had the highest volume of calls
- Which day of the week typically receives the most calls?
- Are there specific times of day that show peak call activity
- Which city (Cincinnati, Cleveland, or Columbus) has the highest total call volume or total amount
- Which representative (R01, R02, etc.) has the top rating
